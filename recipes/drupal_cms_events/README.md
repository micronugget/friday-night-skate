# Drupal CMS Events

Adds an event content type and listing page.

## Recipes Applied
- Document media type (core)
- [Drupal CMS Basic page](https://www.drupal.org/project/drupal_cms_page)
- [Drupal CMS Basic Privacy](https://www.drupal.org/project/drupal_cms_privacy_basic)

## Modules and Themes Installed
- [Add Content by Bundle](https://www.drupal.org/project/add_content_by_bundle)
- [Address](https://www.drupal.org/project/address)
- [Add to Calendar Date Augmenter](https://www.drupal.org/project/addtocal_augment)
- Custom Menu Links (core)
- [Date Augmenter API](https://www.drupal.org/project/date_augmenter)
- [Geocoder](https://www.drupal.org/project/geocoder)
- [Geofield](https://www.drupal.org/project/geofield)
- [Leaflet](https://www.drupal.org/project/leaflet)
- [Smart Date](https://www.drupal.org/project/smart_date)

## Installation
This project is part of [Drupal CMS](https://drupal.org/project/cms). If you're not using Drupal CMS, you can still use this recipe in your own project. Add it with the following Composer command:

```
composer require drupal/drupal_cms_events
```
